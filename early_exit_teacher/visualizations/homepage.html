Methodology & Metrics

The visualization dashboard explores two generation modes that control how the model processes information: Normal Mode (standard full-depth processing through all layers) and Frozen Residual Mode (early exit with frozen residual stream after exiting).

For each mode, we evaluate generation quality using multiple metrics: coherence, clarity, completeness, and no-repetition. These metrics help us understand how early exiting affects the quality of generated text.

The visualizations use color-coded tokens to show which layer each token exited at - red indicates early exits, while blue indicates late exits. We test with different KL divergence thresholds that control how aggressively the model exits early, allowing you to see the trade-off between computational efficiency and generation quality.

Interactive dashboards display layer-by-layer statistics, showing the distribution of exit decisions across different layers and configurations. This reveals patterns in how models decide when they have "enough" information to generate the next token, providing insights into the model's confidence and computational requirements for different types of content.
